# Logo carousel assets

Upload logo files here if you want the home page carousel logos to be stored with the rest of the website images.

Recommended files:

- mohre.webp
- tadbeer.webp
- ajman-chamber.webp
- uae-pass.webp

Current home page also supports the same file names from:

- public/authority-logos/

So either folder will work, but this folder is preferred for new uploads.

Recommended size: 300x120 pixels, WebP, transparent background where possible.
